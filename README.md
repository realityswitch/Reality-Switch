# Reality Switch

This repository contains the prototype planning, systems design,
and early implementation notes for Reality Switch — a moral choice
driven, identity-based game built in Unreal Engine 5.

## Current Goal
Build a first playable vertical slice with:
- One playable area
- One complete mission
- A functioning reputation system
- Visible world reaction

Status: Pre-production
